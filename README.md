# AKS Deployment using Terraform & CI/CD

This repository demonstrates Infrastructure as Code (IaC) for deploying
Azure Kubernetes Service (AKS) using Terraform and automated CI/CD pipelines.

## Tech Stack
- Azure Kubernetes Service (AKS)
- Terraform
- CI/CD (Azure DevOps / GitHub Actions)

## Key Highlights
- Automated AKS provisioning using Terraform
- Provider version locking with terraform.lock.hcl
- Backend configuration for remote state
- Modular and reusable Terraform structure
